# Sunburst Dashboard

Интерактивная аналитическая панель с sunburst графиком на D3.js.

## Функциональность

- Интерактивный sunburst график с анимацией zoom
- Детализация данных при клике на сегменты
- Адаптивный дизайн
- Сброс zoom к корневому виду

## Просмотр
